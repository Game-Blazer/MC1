常用指令

使用说明：以下指令均为服务器控制台指令（无需斜杠），若在游戏内使用需要在指令前添加 / 符号。

死亡不掉落

gamerule keepInventory true
开启后玩家死亡时不会掉落物品和经验

禁用昼夜更替

gamerule doDaylightCycle false
固定游戏时间，不会出现白天和黑夜的交替

禁用自然生命恢复

gamerule naturalRegeneration false
玩家无法通过饥饿值恢复生命值，需要通过其他方式回血

禁用生物生成

gamerule doMobSpawning false
禁止游戏中自然生成怪物和动物

禁用方块掉落

gamerule doTileDrops false
破坏方块时不会掉落物品（如挖矿不会掉矿石）

开启雷电天气

weather thunder
将天气切换为雷暴天气，持续时间默认30秒
重要提醒：使用指令前请确保拥有管理员权限(op)，部分指令需要重启服务器才能生效。

注意：前面加#代表注释（不执行），修改配置后需要保存并重启服务。 
openjdk8="/usr/bin/jdk/jdk1.8.0_361/bin/java"

openjdk17="/usr/bin/jdk/jdk-17.0.6/bin/java"
openjdk19="/usr/bin/jdk/jdk-19.0.2/bin/java"
openjdk21="/usr/bin/jdk/jdk-21.0.2/bin/java"

# Fabric服务端安装核心 
# ${openjdk17} -jar [服务端].jar server -downloadMinecraft

# Fabric服务端启动命令
# ${openjdk17} -Xms1024M -Xmx${SERVER_MEMORY}M -jar [服务端].jar -nogui

安装核心命令

Fabric服务端安装核心命令
${openjdk17} -jar [服务端].jar server -downloadMinecraft
参数说明：
server：指定以服务器模式安装
-downloadMinecraft：自动下载对应的Minecraft服务端文件
${openjdk17}：根据MC版本调整Java版本

Fabric服务端

Fabric服务端信息
链接： Fabric服务端
版本说明：Fabric提供了通用安装器，可选择对应MC版本进行安装，体积更小，启动速度更快。

启动命令

${openjdk17} -Xms1024M -Xmx${SERVER_MEMORY}M -jar [服务端].jar -nogui
参数说明：
-Xms：设置初始内存（建议1024M或更高）
-Xmx：设置最大内存（根据服务器配置调整）
-nogui：禁用图形界面，适合服务器运行

Forge服务端

Forge服务端信息
链接： Forge服务端
版本说明：Forge是最主流的MC模组加载器，支持海量模组，需选择对应MC版本下载安装包。

启动命令

${openjdk17} -Xms1024M -Xmx${SERVER_MEMORY}M -server @libraries/net/minecraftforge/forge/1.17.1-37.1.1/unix_args.txt "$@" -nogui
参数说明：
-Xms：设置初始内存（建议1024M或更高）
-Xmx：设置最大内存（根据服务器配置调整）
-server：启用服务器模式优化
"$@"：传递所有命令行参数给服务端
@libraries/net/minecraftforge/forge/1.17.1-37.1.1/unix_args.txt "$@"：自动生成的run.sh，复制类似的参数(最前面的java不用复制)
